## Installation

### Using `uv` (Python package manager)

1. Install `uv` (Python package manager):

   ```bash
   curl -LsSf https://astral.sh/uv/install.sh | sh
   ```

2. Install dependencies:

   ```bash
   uv sync
   ```

3. Run:

   ```bash
   uv run main.py
   ```

### Main Dependencies

```plain
numpy
matplotlib
pandas
```
